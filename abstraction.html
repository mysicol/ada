abstraction

This is a calculation we could use to find the average of 4 and 6. But what if we want to perform this calculation for any number?

Great! Now, we can compute the average of any numbers we give this function! But what if we want to average things that are not numbers?

Think about it: How would you decide to compute the average of two words? Or two groups of numbers? Being able to use the same calculation in different ways is an important concept in Computer Science called abstraction.
